# Типовые рабочие процессы

## Процесс 1: Оценка нового локомотива

1. Откройте вкладку «🚂 Оценка технического уровня»
2. Выберите тип подвижного состава (например, «Пассажирские электровозы»)
3. Введите данные в форму:
   - Название: «Новый локомотив»
   - Макс. скорость: 180 км/ч
   - Ресурс: 1300 тыс. км
   - Мощность: 8500 кВт
   - Надёжность: 9.5
4. Нажмите «Добавить образец»
5. Нажмите «Рассчитать технический уровень»
6. Посмотрите результаты на диаграммах

## Процесс 2: Прогноз выручки на завтра

1. Откройте вкладку «📈 Прогноз выручки»
2. Проверьте метрики модели (R², MAE)
3. Введите прогнозные значения:
   - Цена топлива: 48 руб./л
   - Протяжённость маршрута: 150 км
   - Праздник: 0
   - Количество поездов: 35
   - Погода: 1
4. Нажмите «Получить прогноз»
5. Запишите полученное значение

## Процесс 3: Оптимизация распределения локомотивов

1. Откройте вкладку «🛤️ Оптимизация перевозок»
2. Проверьте матрицу затрат
3. Проверьте запасы депо и потребности направлений
4. Нажмите «Оптимизировать распределение»
5. Изучите результаты:
   - Минимальные затраты
   - Таблица распределения
   - Тепловая карта
6. При необходимости скорректируйте данные

## Процесс 4: Сравнение нескольких образцов

1. Добавьте несколько образцов через форму
2. Или загрузите CSV с несколькими строками
3. Нажмите «Рассчитать технический уровень»
4. Сравните образцы:
   - На радиальной диаграмме — по каждому параметру
   - На столбчатой диаграмме — по общему рейтингу
