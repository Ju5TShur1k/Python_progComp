# Частые ошибки и их решение

## Ошибка: «Не удаётся загрузить данные с GitHub»
**Причина:** Отсутствует интернет-соединение или недоступен GitHub
**Решение:**
1. Проверьте подключение к интернету
2. Используйте локальный файл Bus.csv (поместите в папку проекта)
3. Проверьте ссылку в config.py

## Ошибка: «Нет данных для отображения»
**Причина:** В выбранной категории нет образцов
**Решение:**
1. Добавьте образец через ручной ввод
2. Загрузите CSV-файл с данными
3. Переключитесь на другую категорию

## Ошибка: «Суммы запасов и потребностей должны быть больше 0»
**Причина:** В полях оптимизации нулевые значения
**Решение:**
1. Убедитесь, что все запасы > 0
2. Убедитесь, что все потребности > 0

## Ошибка: Модель не обучается
**Причина:** Некорректные данные
**Решение:**
1. Проверьте формат данных в Bus.csv
2. Убедитесь, что все колонки заполнены

## Кнопка «Получить прогноз» не даёт результат
**Причина:** Некоторые поля не заполнены
**Решение:**
1. Заполните все поля ввода
2. Убедитесь, что значения в допустимом диапазоне
