# Словарь терминов (железнодорожная тематика)

## Термины интерфейса
**ИАС** — Информационно-аналитическая система
**Dash** — Фреймворк для создания веб-интерфейса
**Callbacks** — Обработчики событий (реагируют на действия пользователя)

## Термины подвижного состава
**Электровоз** — Локомотив, работающий от электричества
**Тепловоз** — Локомотив с двигателем внутреннего сгорания
**Депо** — Место хранения и обслуживания локомотивов
**Направление** — Железнодорожный маршрут между станциями

## Термины прогнозирования
**R²** — Коэффициент детерминации (качество модели, 0-1)
**MAE** — Средняя абсолютная ошибка (точность прогноза)
**Полиномиальная регрессия** — Модель, учитывающая нелинейные зависимости
**Признаки** — Входные параметры для прогноза

## Термины оптимизации
**Транспортная задача** — Оптимизация распределения ресурсов
**Симплекс-метод** — Алгоритм решения задач линейного программирования
**Затраты** — Стоимость обслуживания (руб./км)
