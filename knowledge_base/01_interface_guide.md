# Руководство по интерфейсу ИАС Железнодорожного перевозчика

## Главное меню
Приложение состоит из трёх основных вкладок:

### 1. 🚂 Оценка технического уровня
- **Назначение:** Оценка качества подвижного состава
- **Элементы:**
  - Выпадающий список «Выбор типа подвижного состава»
    - Пассажирские электровозы
    - Грузовые тепловозы
    - Электропоезда (пригородные)
  - Кнопка «Рассчитать технический уровень»
  - Поля для ручного добавления образца
  - Загрузка CSV-файла
  - Радиальная диаграмма
  - Столбчатая диаграмма

### 2. 📈 Прогноз выручки
- **Назначение:** Прогнозирование финансовых показателей
- **Элементы:**
  - Информация о модели (R², MAE)
  - Поля для ввода признаков:
    - Цена топлива (руб./л)
    - Протяжённость маршрута (км)
    - Праздник/выходной (0/1)
    - Количество поездов
    - Погода (0/1)
  - Кнопка «Получить прогноз»
  - График зависимости выручки

### 3. 🛤️ Оптимизация перевозок
- **Назначение:** Оптимальное распределение локомотивов
- **Элементы:**
  - Матрица затрат (руб./км)
  - Запасы локомотивов в депо
  - Потребность направлений
  - Кнопка «Оптимизировать распределение»
  - Таблица результатов
  - Тепловая карта

## Общие элементы управления
- **Загрузка CSV:** Перетащите файл или нажмите для выбора
- **Кнопки:** Выполняют действия (расчёт, прогноз, оптимизация)
- **Графики:** Интерактивные, можно приближать, скачивать
