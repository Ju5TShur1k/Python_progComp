# Клавиатурные сокращения и советы

## Навигация
- **Ctrl + 1** — Вкладка «Оценка технического уровня»
- **Ctrl + 2** — Вкладка «Прогноз выручки»
- **Ctrl + 3** — Вкладка «Оптимизация перевозок»

## Работа с графиками
- **Наведение** — просмотр точных значений
- **Двойной клик** — сброс масштаба
- **Перетаскивание** — выделение области для приближения
- **Скачивание** — кнопка камеры в правом верхнем углу графика

## Полезные советы
1. **Перед прогнозом** всегда проверяйте качество модели (R² > 0.9 — хорошо)
2. **Для точной оптимизации** убедитесь, что сумма запасов = сумме потребностей
3. **При добавлении образцов** заполняйте все поля
4. **Сохраняйте важные данные** экспортом графиков

## Быстрые действия
- **Обновить данные** — перезапустите приложение (Ctrl+C в консоли, затем python main.py)
- **Очистить чат** — перезапустите чат-помощник
